# plugin-rvm

Plugin for [Oh My Fish][omf-link] that integrates fish with [rvm][rvm-link].

It uses the official approach suggested [here][rvm-fish-link].


## Install

```fish
$ omf install plugin-rvm
```


## Usage

```fish
$ rvm help
```
